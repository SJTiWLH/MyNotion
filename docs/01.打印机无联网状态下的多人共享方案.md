# 打印机无联网状态下的多人共享方案

当打印机无法联网时，可通过「主机有线连接+局域网共享」实现多人使用，老旧打印机也适用。

## 一、主设备（连接打印机的电脑）设置

1. **基础连接测试**
    
    用USB数据线连接打印机与主机，确保主机可正常打印（建议打印测试页验证）。
    
2. **开启打印机共享**
    
    - 找到「控制面板 → 设备和打印机」，右键目标打印机 → 「打印机属性」。
    - 切换至「共享」选项卡，勾选「共享这台打印机」，可自定义共享名称。
3. **配置网络共享权限**
    
    - 打开「控制面板 → 网络和共享中心 → 更改高级共享设置」。
    - 启用以下选项：
        - 网络发现（选择「启用网络发现」）
        - 文件和打印机共享（选择「启用文件和打印机共享」）
        - 密码保护（选择「关闭密码保护共享」）。
    - 保存设置。
4. **记录主机信息**
    
    - 打开「此电脑 → 右键属性」，复制「设备名称」（如`DESKTOP-XXX`），供其他设备连接使用。

## 二、其他设备（需共享打印机的电脑）设置

1. **添加Windows凭证**
    
    - 打开「控制面板 → 凭证管理器 → Windows凭证 → 添加Windows凭证」。
    - 网络地址填写：`\\设备名称`（例如`\\DESKTOP-XXX`），用户名：GUEST /密码为空（无密码可留空）。
2. **确保网络一致**
    
    连接与主机相同的WiFi或局域网。
    
3. **访问共享打印机**
    
    - 按`Win + R`，输入`\\设备名称`（如`\\DESKTOP-XXX`），回车。
    - 右键弹出的共享打印机 → 「连接」，按提示安装驱动即可。

## 完成效果

其他设备会显示连接的打印机名称：`{设备名称}上的{打印机名称}`，可直接用于打印。
