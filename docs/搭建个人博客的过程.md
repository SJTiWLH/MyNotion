# VitePress + Teek 主题 个人博客搭建 & Cloudflare 完整部署记录

## 一、项目介绍
本次个人博客采用 **VitePress** 静态站点框架，搭配高颜值 **Teek 主题** 进行搭建。
Teek 主题文档：[https://vp.teek.top/](https://vp.teek.top/)

整体实现效果：
- 高颜值全屏壁纸首页、打字机标语、动态界面
- 文章置顶、精选卡片、时间线、分类标签
- 完美适配 Obsidian Markdown 笔记
- 全自动云端部署、公网可访问、绑定个人免费域名

## 二、技术栈
- 框架：VitePress `v1.6.4`
- 主题：vitepress-theme-teek（高颜值博客主题）
- 代码托管：GitHub
- 部署平台：Cloudflare Pages
- 域名服务：my.dnshe.com
- 本地写作：Obsidian

## 三、搭建流程记录

### 1. 本地项目搭建
1. 初始化 VitePress 项目
2. 安装并配置 `vitepress-theme-teek` 主题
3. 配置主题 Banner 全屏壁纸、首页文案、动画效果
4. 配置文章置顶、精选、排序、标签分类
5. 整理 Obsidian 笔记，统一迁移为博客文章格式

### 2. 核心特色功能实现
通过 Teek 主题配置，实现了博客常用功能：
- 首页全屏动态壁纸背景
- 打字机/切换式首页简介
- 文章**精选卡片展示** + **文章列表同时展示**
- 文章置顶排序、时间自动排序
- 页面过渡动画、加载动画
- 完全兼容 Markdown 所有语法

### 3. 代码托管至 GitHub
将本地完整博客项目上传至 GitHub 远程仓库：
- 维护完整版本管理
- 作为 Cloudflare Pages 自动部署源
- 支持随时更新笔记、自动发布上线

### 4. Cloudflare Pages 云端部署
1. 关联 GitHub 博客仓库
2. 配置 VitePress 自动构建命令
3. 开启云端自动构建、自动部署
4. 解决图片路径、静态资源、构建报错问题
5. 清除缓存、修复主题部署兼容问题

### 5. 绑定免费公网域名
最终成功配置 Cloudflare 免费域名，
实现 **全网公网访问个人博客**，无需服务器、零成本托管。

## 四、最终成果
- ✅ 基于 VitePress + Teek 主题的高颜值个人博客
- ✅ 所有笔记在线公开、结构化展示
- ✅ GitHub 一键更新、Cloudflare 自动部署
- ✅ 免费公网域名、全球 CDN 加速访问
- ✅ 完美适配日常笔记写作与技术沉淀

## 五、后续维护方式
1. 本地 Obsidian 写笔记
2. 本地调试无误后推送 GitHub
3. Cloudflare 自动拉取代码、自动构建、自动更新网站
4. 通过 FrontMatter 自由控制文章置顶、精选、排序、封面
